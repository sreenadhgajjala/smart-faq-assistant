# 🤖 Smart FAQ Assistant

An AI-powered FAQ Chatbot built using **Python, Streamlit, NLP, TF-IDF, and Cosine Similarity**. The chatbot answers user questions by matching them with a predefined FAQ knowledge base related to Artificial Intelligence, Machine Learning, Deep Learning, and NLP.

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash

cd smart-faq-assistant
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment:

**Windows**





## 📖 How It Works

1. FAQs are stored in a knowledge base.
2. User enters a question.
3. Text is preprocessed using NLP techniques.
4. TF-IDF converts text into vectors.
5. Cosine Similarity finds the closest matching FAQ.
6. The chatbot returns the most relevant answer.
